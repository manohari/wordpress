Contributor: Manohari Vagicherla

Tags: blog, import, pages,import blog page
Requires at least: 2.8.5


Blog page import tool for all blogs 

== Description ==
This Plugin imports any blog page content inot your wordpress post. It takes URL as input and retreive total content.In version 1 we imported the blog page as it is i.e., it import the header and footer of your blog
== Installation ==

1. Upload the import_blog folder to your wp-content/plugins folder
2.Activate the plugin
3. Go to Tools/Import you will find a link with name Import Page From Blog



== Frequently Asked Questions ==

= How to use it? =

1. Input the URL from which you want get links
2. Click on "get Content".
3. Your content will be added as post in your blog.The completed messages will be shown on the same. Thats it

= How to uninstall it? =

Just delete it from wp-content/plugin. In fact it does no changes to the database or other core codes.
